Що таке CSS?

CSS (Cascading Style Sheets) — це мова, яка описує, як елементи HTML мають виглядати на екрані. CSS повністю відповідає за колір, шрифт, розміри та розташування всіх елементів.

Зверніть увагу, що цей абзац має інший колір тексту та рамку, тому що ми застосували до нього спеціальний клас "важлива_інформація".

Основні елементи, які ми оформили:

  1. Заголовки (h1, h2, h3)
  2. Фон сторінки (body)
  3. Список меню (ul, li)
  4. Абзаци тексту (p)
